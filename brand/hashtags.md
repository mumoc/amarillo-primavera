# Banco de Hashtags

Lista de referencia para no improvisar hashtags en cada post. Se puede ampliar conforme se prueben nuevos.

## Generales de marca (usar casi siempre)

`#AmarilloPrimavera` `#HechoAMano` `#Artesanal` `#Colima` `#ProductosArtesanales` `#RegalosArtesanales` `#HechoEnMexico`

**Organizadores**
`#OrganizadoresArtesanales` `#DecoArtesanal`

## Notas de uso

- Combinar 1–2 generales de marca + 2–4 de categoría por post. No saturar con más de 8–10 hashtags.
- Pinterest funciona mejor con descripciones más naturales y menos hashtags explícitos; usarlos solo si aplican.
- Si un producto nuevo no cae en ninguna categoría, agregar su propio bloque aquí en vez de inventar tags sueltos cada vez.
