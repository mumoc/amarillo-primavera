Muñeca Marien vestida de blanco para Primera Comunión, con corona de flores a juego y un ramo de flores de tela. Viene acompañada de una coronita y un ramo extra a juego, ideales para complementar el recuerdo. Una pieza especial para celebrar este día tan importante, hecha a mano con mucho cariño.
