# Organizadores de Colores

Producto trabajado como parte de ejercicios de etiqueta e imagen para producto.

## Categoría

Organizadores.

## Concepto

Organizadores de colores con una presentación moderna, limpia y lista para impresión.

## Dirección visual

Se exploraron etiquetas:

- Modernas.
- Vintage.
- Planas.
- Listas para imprimir.
- Con uso de foto del producto.
- Sin cilindro.
- Con imagen simplificada tipo acuarela.

## Notas visuales

La etiqueta debe funcionar como pieza imprimible y mostrar el producto de forma clara, sin elementos innecesarios.

Debe mantener una composición limpia y directa, especialmente cuando se requiera “solo la etiqueta”.
