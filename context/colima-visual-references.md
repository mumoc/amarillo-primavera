# Referencias Visuales de Colima

Amarillo Primavera tiene una conexión visual con Colima.

## Elementos de referencia

- Árbol primavera amarillo.
- Paisajes de Colima.
- Volcán de Fuego.
- Nevado de Colima.
- Colores vivos y cálidos.
- Naturaleza local.
- Sensación mexicana y artesanal.

## Imagen conceptual trabajada

Se pidió una imagen de un árbol primavera amarillo con los volcanes de Colima al fondo.

Se aclaró que son dos volcanes:

- El Volcán de Fuego.
- El Nevado de Colima.

También se buscó que la imagen fuera más realista y que el árbol se viera un poco más grande.

## Uso para marca

Estas referencias pueden usarse en:

- Banners.
- Fondos de marca.
- Etiquetas.
- Publicaciones de redes sociales.
- Historia visual de Amarillo Primavera.
- Material turístico.
